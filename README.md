# Spam Message Classifier

## Overview

A Python machine learning project that classifies text messages as **spam** or **ham (not spam)** using natural language processing and machine learning techniques. The project demonstrates the complete workflow from text preprocessing to model training and prediction.

## Features

* Clean and preprocess text messages
* Convert text into numerical features
* Train a spam classification model
* Predict whether new messages are spam or ham
* Evaluate model performance

## Technologies

* Python
* Pandas
* scikit-learn
* NumPy

## Skills Demonstrated

* Machine Learning
* Natural Language Processing (NLP)
* Text Preprocessing
* Model Training
* Classification
